## Описание

С помощью методов класса приложения работают с данными кассовой смены.

## Методы

### printZReport

```js
static printZReport(): Promise<void>
```

**Описание**

Печатает Z-отчёт.

**Возвращает**

* `Promise`, результат которого – `void`
